# Scripts

Utility scripts for the CREXI Deal Finder pipeline.

## Planned Scripts

- `validate-listing.js` — Validate listing JSON against schema
- `deduplicate.js` — Remove duplicate listings across data files
- `export-csv.js` — Export scored deals to CSV for spreadsheet analysis
- `calculate-payments.js` — Owner finance payment calculator
- `merge-runs.js` — Merge multiple scrape runs into single dataset
